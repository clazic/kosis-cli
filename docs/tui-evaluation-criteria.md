- 각 항목: 배점 x (0%, 50%, 100%) = 미구현/부분구현/완전구현
- 80점 이상: 통과
- 80점 미만: 수정 후 재평가
- Critical 문제 (크래시, 보안): 해당 항목 0점 + 즉시 수정 필수
