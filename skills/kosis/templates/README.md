# KOSIS 차트 HTML 템플릿

AI가 kosis-cli 데이터를 시각화할 때 사용하는 HTML 템플릿입니다.

## 템플릿 목록

| 템플릿 | 용도 | 차트 타입 |
|--------|------|----------|
| `line-chart.html` | 시계열 추이 (단일/다중 시리즈) | 라인 |
| `comparison.html` | 비교 분석 (차트 + 데이터 테이블 + 비중 계산) | 라인/바 |
| `bar-rank.html` | 항목별 순위 (가로 막대, 자동 정렬) | 바 |
| `bar-vertical.html` | 세로 막대 비교 (다중 시리즈 지원, 라벨 자동 회전) | 바 |
| `stacked-bar.html` | 누적 막대 (구성비 분석, 합계 표시) | 바 |
| `pie-share.html` | 구성비/점유율 (도넛 차트) | 파이 |
| `area.html` | 영역 차트 (면적 그라데이션, 다중 시리즈) | 라인/영역 |
| `dual-axis.html` | 이중 Y축 (좌: 바, 우: 라인, 단위 다른 지표 비교) | 바+라인 |
| `heatmap.html` | 히트맵 (X=시점, Y=분류, 색상 매핑) | 히트맵 |
| `treemap.html` | 트리맵 (비중 계층 표시, 비중% 포함) | 트리맵 |
| `dashboard.html` | 대시보드 (여러 차트를 2열 그리드로 한 페이지에) | 복합 |

## 사용법

### 1단계: kosis-cli로 데이터 조회
```bash
kosis d 101 DT_1IN1502 -c1 "11+21+26" -i T100 -p Y -l 5 -f json > data.json
```

### 2단계: 템플릿의 플레이스홀더를 치환

템플릿 파일을 읽고 아래 플레이스홀더를 치환하여 최종 HTML을 생성합니다.

#### 텍스트 플레이스홀더
| 플레이스홀더 | 설명 | 필수 |
|-------------|------|:---:|
| `{{TITLE}}` | 차트 제목 | O |
| `{{SUBTITLE}}` | 부제목/설명 | O |
| `{{NOTE}}` | 주석 (NOTE_START~NOTE_END 주석 해제 필요) | X |
| `{{SOURCE}}` | 출처 표시 (**통계표 한글 이름** 포함, 아래 규칙 참조) | O |

**`{{SOURCE}}` 작성 규칙:**
- 반드시 **통계표의 한글 이름**을 포함할 것 (검색 결과의 `TBL_NM` 또는 meta에서 확인)
- 형식: `출처: KOSIS 「통계표한글이름」 (ORG_ID/TBL_ID) | kosis-cli`
- 예시: `출처: KOSIS 「인구(읍면동/5세별)」 (101/DT_1IN1502) | kosis-cli`
- 예시: `출처: KOSIS 「시도별미분양현황」 (116/DT_MLTM_2086) | kosis-cli`

#### 데이터 플레이스홀더 (`{{DATA}}`)
JSON 형식으로 치환합니다:

```json
{
  "labels": ["2020", "2021", "2022", "2023", "2024"],
  "unit": "명",
  "series": [
    { "name": "서울특별시", "values": [9586195, 9472127, 9417469, 9384512, 9335444] },
    { "name": "부산광역시", "values": [3349016, 3324335, 3295760, 3279604, 3257256] }
  ],
  "chartType": "line",
  "showRatio": true,
  "markLine": { "xAxis": "2020", "label": "기준 변경" }
}
```

#### DATA 필드 설명
| 필드 | 설명 | 필수 | 사용 템플릿 |
|------|------|:---:|------------|
| `labels` | X축 라벨 (시점 등) | O | 전체 |
| `unit` | 단위 (명, 백만원 등) | O | 전체 |
| `series` | 시리즈 배열 [{name, values}] | O | 전체 |
| `chartType` | 차트 타입 ("line" / "bar") | X | comparison |
| `showRatio` | 마지막 시리즈의 비중(%) 테이블 표시 | X | comparison |
| `markLine` | 기준선 표시 {xAxis, label} | X | comparison |

## AI가 데이터를 변환하는 방법

kosis-cli의 JSON 출력:
```json
[
  {"행정구역별(읍면동)": "서울특별시", "시점": "2024", "수치값": "9335444", "단위": "명", ...},
  {"행정구역별(읍면동)": "부산광역시", "시점": "2024", "수치값": "3257256", "단위": "명", ...}
]
```

→ 템플릿 DATA로 변환:
```json
{
  "labels": ["2024"],
  "unit": "명",
  "series": [
    { "name": "서울특별시", "values": [9335444] },
    { "name": "부산광역시", "values": [3257256] }
  ]
}
```

변환 규칙:
1. 분류 컬럼의 고유값 → `series[].name`
2. 시점 컬럼의 고유값 → `labels`
3. 수치값 → `series[].values` (문자열을 숫자로 변환)
4. 단위 → `unit`
