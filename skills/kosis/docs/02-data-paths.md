# 2. 세 가지 데이터 경로

KOSIS CLI는 용도와 숙련도에 따라 세 가지 데이터 조회 경로를 제공합니다.

## 경로 1: 통계표 (28만개, 세밀한 조회)

```
search → meta → data
(검색)   (코드 확인)  (데이터)
```

각 통계표마다 분류/항목/주기 코드가 다르므로, `meta`로 먼저 확인해야 합니다.

```bash
# 예시: 인구통계 조회 전체 흐름
kosis s "인구"                          # 1단계: 검색으로 통계표 찾기
kosis m 101 DT_1IN1502                  # 2단계: 메타로 분류/항목 코드 확인
kosis d 101 DT_1IN1502 -c1 00 -i T100 -p Y -l 5   # 3단계: 데이터 조회
```

**적합한 경우:** 정밀한 분류/항목/시점 조건이 필요할 때, 대용량 데이터를 파일로 내보낼 때

## 경로 2: 주요지표 (1,473개, 간편 조회)

```
ind search → ind data
(지표명 검색)  (수치 바로 조회)
```

지표명만 알면 코드 없이 바로 수치 조회 가능합니다.

```bash
# 예시: GDP 지표 간편 조회
kosis ind s "GDP"                       # 지표 검색
kosis ind d "GDP"                       # 수치 바로 조회
kosis ind d "소비자물가" -n 20 -o 물가.xlsx  # 최근 20개 + Excel 저장
```

**적합한 경우:** 주요 경제/사회 지표를 빠르게 확인할 때, 코드를 몰라도 지표명으로 조회할 때

## 경로 3: 자연어 원스텝 (가장 간단)

```
quick "자연어 요청"
→ 규칙 매칭 또는 AI로 자동 변환 → 데이터 출력
```

통계표 코드를 몰라도 자연어로 바로 조회합니다.

```bash
# 예시: 자연어로 한 줄 조회
kosis q "서울 미분양 최근 5년"
kosis q "GDP 2020~2024"
kosis q "인구 2015,2020,2025"
kosis q "실업률 최근 12개월" --ai claude   # AI 활용
```

**적합한 경우:** CLI에 익숙하지 않을 때, 탐색적으로 빠르게 데이터를 확인할 때

## 경로 선택 가이드

| 상황 | 추천 경로 | 이유 |
|------|----------|------|
| 처음 사용, 코드를 모름 | 경로 3 (quick) | 자연어만으로 조회 가능 |
| 주요 경제지표 확인 | 경로 2 (indicator) | 지표명으로 간편 조회 |
| 세밀한 조건 지정 | 경로 1 (통계표) | 분류/항목/시점 정밀 제어 |
| 대용량 데이터 내보내기 | 경로 1 (통계표) | 자동 분할 + 파일 저장 지원 |
