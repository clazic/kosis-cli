# 11. 연관 통계표 찾기

같은 종류의 데이터가 연도별/구분별로 분리된 통계표를 찾는 방법입니다.

## 방법 1: STAT_ID로 같은 조사의 표 찾기

`search` 결과의 `STAT_ID`(조사ID)가 같으면 같은 통계조사의 관련 표입니다.

```bash
kosis s "인구"
# → STAT_ID가 같은 표들 = 같은 조사의 형제 표
```

## 방법 2: 계층 탐색으로 형제 표 찾기

```bash
kosis ls --parent A_4        # 인구총조사 하위
kosis ls --parent A11        # 인구부문 → 연도별 표 목록
# → A11_2015_1: 총조사인구(2015년 이후)
# → A11_2010:   총조사인구(2010년)
# → A11I:       총조사인구(2005년)
```

## 방법 3: 키워드 검색

```bash
kosis s "미분양" -n 20       # 유사 주제의 다른 기관/조사 표 발견
```

**한계:** KOSIS API에 "연관 통계표 조회" 전용 API는 없습니다. 찾은 표들의 분류체계/코드가 동일한지는 `meta`로 개별 확인이 필요합니다.
