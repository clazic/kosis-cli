# 4. 주요지표 조회 — `indicator` (별칭: `ind`)

KOSIS 주요지표는 통계청이 선별한 1,473개의 핵심 지표로, 통계표 코드 없이 지표명만으로 간편하게 조회할 수 있습니다.

## 4.1 지표 검색

```bash
kosis ind search "GDP"          # 또는 kosis ind s "GDP"
kosis ind s "실업률" -n 20      # 최대 20개 결과
kosis ind s "인구"              # 인구 관련 지표 검색
kosis ind s "물가"              # 물가 관련 지표 검색
```

검색 결과에는 지표ID, 지표명, 단위, 주기 등이 표시됩니다.

## 4.2 지표 상세 정보

```bash
kosis ind info 160              # 지표ID로 조회 (개념, 산식, 출처)
```

상세 정보에 포함되는 항목:
- **개념 정의**: 지표의 정의와 의미
- **산출 방식**: 계산 공식 또는 산출 근거
- **출처**: 작성 기관 및 통계명
- **수록 주기**: 연/분기/월 등
- **최신 수록 시점**: 가장 최근 데이터 시점

## 4.3 지표 수치 조회

```bash
kosis ind data "GDP"            # 또는 kosis ind d "GDP"
kosis ind d "소비자물가" -n 20 -f json   # 최근 20개, JSON 출력
kosis ind d "실업률" -o 실업률.xlsx       # Excel 파일 저장
kosis ind d "GDP" -f csv                  # CSV 형식 출력
kosis ind d "경제성장률" -n 10            # 최근 10개 시점
```

| 플래그 | 설명 | 기본값 |
|--------|------|--------|
| `-n` | 조회할 시점 수 | 10 |
| `-f` | 출력 형식 (table/json/csv) | table |
| `-o` | 파일 저장 (xlsx/csv/db/parquet) | - |

## 4.4 지표 목록 탐색

계층 구조로 지표를 탐색할 수 있습니다.

```bash
kosis ind list                  # 최상위 목록
kosis ind ls I01                # 하위 목록 탐색
kosis ind ls I01_01             # 더 하위 목록
```

## 4.5 통계표 경로와의 차이점

| 구분 | 통계표 (`data`) | 주요지표 (`ind`) |
|------|----------------|-----------------|
| 대상 수 | 28만개 | 1,473개 |
| 코드 필요 | ORG_ID + TBL_ID 필수 | 지표명으로 조회 가능 |
| 세밀한 조건 | 분류/항목/시점 자유 설정 | 사전 정의된 조건 |
| 파일 저장 | 모든 형식 지원 | 모든 형식 지원 |
| 적합 용도 | 정밀 분석 | 빠른 현황 파악 |
