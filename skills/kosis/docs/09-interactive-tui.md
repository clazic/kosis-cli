# 9. 대화형 모드 & TUI 대시보드

## 9.1 대화형 모드

필수 인자 생략 시 단계별 안내가 시작됩니다. 완료 후 재사용 가능한 원라인 명령어가 표시됩니다.

```bash
kosis data     # 검색 → 선택 → 파라미터 설정 → 조회 → 내보내기
kosis search   # 검색어 입력 대기
kosis meta     # 검색 → 선택 → 메타 표시
kosis explain  # 검색 → 선택 → 조사 설명
```

**`kosis data` 대화형 흐름:**

```
1. 검색어 입력: 인구
2. 통계표 선택: [0] 인구(읍면동/5세별) (101/DT_1IN1502)
3. 분류값 선택: [0] 전국 (00)
4. 항목 선택:   [0] 총인구 (T100)
5. 수록주기:    Y (연)
6. 시점:        최근 5개
→ 데이터 출력
→ 내보내기 (y/n)?
→ 원라인 명령어: kosis d 101 DT_1IN1502 -c1 "00" -i T100 -p Y -l 5
```

> 입력 중 `q` 또는 `quit` 입력 시 종료됩니다. "예/아니오", "네/ㅇ/ㄴ" 등 한글 응답도 지원합니다.

## 9.2 TUI 대시보드

```bash
kosis          # 인자 없이 실행 → TUI 대시보드 진입
```

| 키 | 동작 |
|----|------|
| `/` | 검색 |
| `↑↓` | 탐색 |
| `Enter` | 선택/조회 |
| `Tab` | 패널 전환 (검색→메타→파라미터→결과) |
| `e` | 내보내기 |
| `f` | 즐겨찾기 |
| `i` | 통계표/지표 모드 전환 |
| `?` | 도움말 |
| `q` | 종료 |
